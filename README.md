# DoneWizIt Media (Website)

This repository contains only the **web app** for DoneWizIt media (playlist selector and audio player). Legacy Expo / React Native mobile app code has been removed.

## Quick start

From the repo root:

```bash

# first time ... install packages 
cd web && npm install
```

```bash
# next time 
cd web && npm run dev

# test 
npm run build
```

This runs the Vite dev server from the `web` directory.

For setup, environment variables, assets, and Vercel deployment, see **[web/README.md](web/README.md)**.
